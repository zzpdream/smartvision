### Git常用命令

1. 从服务器上拉取最新的代码

		git pull origin master
		
2. 向服务器提交最新的代码

		git push origin master
		
3. 将本地修改提交到暂存区

		git add filename
		
4. 将本地暂存区的更新提交到本地仓库

		git commit -m 'commit message'
		
5. 创建新的分支

		git checkout -b name
		
		or
		
		git branch name
		git checkout name
